About the Game

Market Mayhem is a cozy yet frantic co-op game where coordination is key. Players must work together to deliver fresh fruits from a delivery truck to various market stalls in a bustling town square.

Racing against a strict time limit, players need to manage inventory, navigate obstacles, and ensure every stall gets their daily produce.

Technical Contributions

  • Developed smooth player movement and character animation logic
  • Created the central delivery system logic using state machines
  • Built the UI for timers and scoring feedback loops
