About the Game

Fish-Tick is a chaotic 2-player local multiplayer fishing game. Originally designed for touch-screen tables, it has been optimized for mobile devices to bring the fun anywhere.

Players compete to catch the most fish in a frantic environment where chaos is guaranteed. The game features vibrant cartoon graphics and simple, addictive gameplay mechanics.

Technical Contributions

  • Implemented randomized fish spawning logic
  • Developed touch input and tap mechanics for mobile optimization
  • Built the entire user interface using Unity Canvas system
